DOM이란?

DOM(Document Object Model : 문서 객체 모델)은 HTML 문서를 읽고 조작하는 객체의 집합이다. 즉 HTML 문서의 구조를 말하며, 자바 스크립트를 사용하여 문서객체를 선택하고, 속성 또는 스타일 을 적용한다.

jQuery란?

HTML DOM(Document Object Model)과 자바스크립트 사이의 상호작용을 간단하게 해주는 오픈소스 자바스크립트 라이브러리이다. '자바스크립트(j)에서 DOM 엘리먼트를 질의(query)한다.' 이 문장에서 j와 query를 합친 단어이다. jQuery함수는 jQuery(); 또는 $(); DOM에서 엘리먼트(요소, 태그)를 선택하기 위해서 사용한다. jQuery 래퍼(wrapper)라고도 불리며, 메소드의 매개변수로 CSS 셀렉터 표현식을 문자열 형태로 기술한다. $(document).ready(); ==> html문서가 웹브라우저에 로드되어서 준비된 상태가 되었을 때 적당한 처리를 하기 위한 이벤트 핸들러이다.
